📋 Quy tắc xử lý:

🔸 Xử Lý Format:
  • Đánh dấu không-logic: Giờ ra < 12:00 hoặc giờ ra < giờ vào
  • Swap: Nếu giờ ra < giờ vào → hoán đổi 2 giá trị
  • Fix bug: Sau khi swap, nếu giờ ra vẫn < 12:00 → +1 giờ làm
🔸 Xử Lý File Tách:
  • Cell rỗng: Giờ làm và tăng ca = 0
  • Giờ < 5:00: Tự động +12 tiếng (0:30 → 12:30)
  • Thứ 2,4,6: Giờ ra 19:00-19:15 nếu > 17:30
  • Thứ 3,5,7,CN: Giờ ra 17:30-17:35 nếu > 17:30
🔸 Xử Lý Giờ Tăng Ca:
  • Input: Cần 2 files (File Gốc + File Đã Tách)
  • Ngày thường: Final = |Gốc - Đã tách|
  • Ngày CN: Final = Copy nguyên từ Đã tách, Đã tách mới = 0
  • Output: File Đã Tách Mới (không CN) + File Final (có CN + diff)
